\begin{abstract}
\noindent
\textbf{v0.3.0 update (hypothesis shift):}
This version pivots from the earlier ``resonant space'' direction (v0.2.x) to a
topological-defect picture.
We introduce a \emph{dimensionless} scalar field $\EnchanS$ interpreted as an effective
time-dilation factor (a compact proxy for gravitational time delay).

Assuming that $\EnchanS$ admits logarithmic defect-like profiles anchored by baryonic structure,
we show how an effective acceleration closure compatible with the Radial Acceleration Relation (RAR)
and the Baryonic Tully--Fisher Relation (BTFR) can emerge at the phenomenological level.
A central goal of v0.3.0 is to express the characteristic acceleration scale $a_0$
as a derived parameter in terms of vacuum stiffness and defect geometry, rather than as an
unexplained constant.
\end{abstract}

\section*{Status and Scope (v0.3.0)}
This document presents a \textbf{speculative effective framework} intended to sit
beneath (not replace) the Standard Model and General Relativity.
\begin{itemize}
    \item \textbf{Empirical baseline:} The included Python scripts reproduce standard
    galactic scaling regularities (SPARC-based RAR / BTFR / fixed-parameter rotation-curve prediction)
    as external benchmarks. Verification code is available at:\\
    \url{https://github.com/EnchanTheory/Enchan-Field-Notes}
    
    \item \textbf{Theoretical proposal:} The working hypothesis is that these regularities can be
    interpreted as signatures of defect-like configurations in the field $\EnchanS$.
    \item \textbf{Assumptions:} The ``Anchor Condition'' linking baryonic structure to the defect scale
    is currently a phenomenological ansatz and is explicitly marked as such.
    \item \textbf{Conservativeness:} The framework is not presented as a modification of strong-field GR;
    it targets the galaxy-scale, weak-field phenomenology where the empirical relations are defined.
\end{itemize}
